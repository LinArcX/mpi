## Installation
Just copy `mpi.vim` into your `plugin` directory.

## Costumization
You can add [unicode emojies](https://en.wikipedia.org/wiki/Template:Emoji_(Unicode_block)) like this:

```
let g:mpi#exact_matches = {
    ...
    \ '.zshrc'                           : 'ğŸŒ',
    \ '.zsh_history'                     : "\<Char-0x23f0>",
    \ '.fehbg'                           : 'ğŸˆ',
    ...
\}
```
